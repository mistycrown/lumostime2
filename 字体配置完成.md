## 添加新字体

以后要添加新字体，只需 3 步：

### 步骤 1：放置字体文件
将 `.woff2` 文件放到 `public/fonts/` 目录

### 步骤 2：在 `public/fonts/fonts.css` 中添加定义
```css
@font-face {
    font-family: '你的字体名称';
    src: url('./你的字体文件.woff2') format('woff2');
    font-weight: 400;
    font-style: normal;
    font-display: swap;
}
```

### 步骤 3：在 `src/services/fontService.ts` 中添加配置
```typescript
{
    id: 'your-font-id',
    name: 'your-font-id',
    displayName: '你的字体名称',
    description: '字体描述',
    fontFamily: '"你的字体名称", "Noto Serif SC", serif',
    type: 'custom'
}
```

就这么简单！

## 注意事项

- 字体文件必须是 `.woff2` 格式
- 文件名在 CSS 中必须完全匹配
- `fontFamily` 中的字体名称要和 `@font-face` 中的 `font-family` 一致
- 建议添加回退字体（如 `"Noto Serif SC", serif`）
