# UI Icon Manager - 图标管理工具

## 功能说明

这是一个可视化的图标管理工具，可以让你轻松地交换和重新排列图标文件。

## 使用方法

### 1. 启动服务器

在项目根目录运行：

```bash
python icon_manager.py
```

服务器会自动启动并在浏览器中打开管理界面（地址：http://localhost:8888）

### 2. 选择主题

- 页面顶部会显示所有可用的图标主题
- 点击主题按钮切换到对应的主题

### 3. 交换图标

#### 步骤：

1. **开启交换模式**：点击"开启交换模式"按钮
2. **选择第一个图标**：点击你想交换的第一个图标（会显示绿色边框）
3. **选择第二个图标**：点击你想交换的第二个图标
4. **自动交换**：系统会自动交换这两个图标的文件名
5. **继续交换**：可以继续选择其他图标进行交换

#### 示例场景：

如果你发现：
- 78号图标（lock/锁定）和 79号图标（star/星标）的图片放错了
- 只需在交换模式下依次点击 #78 和 #79
- 系统会自动交换这两个文件，刷新后就能看到正确的顺序

### 4. 验证结果

交换完成后：
- 页面会自动刷新显示新的图标顺序
- 右上角会显示成功提示
- 你可以在 `icon-checker.html` 中验证所有主题的图标是否正确

## 工作原理

1. **Python 服务器**：提供 HTTP 服务和文件操作 API
2. **Web 界面**：可视化显示所有图标，支持点击交换
3. **文件交换**：直接修改 `public/uiicon/{theme}/` 目录下的文件名

## 注意事项

⚠️ **重要提示**：
- 交换操作会直接修改文件系统中的图标文件
- 建议在交换前备份 `public/uiicon/` 目录
- 一次只能交换一个主题的图标
- 交换后需要刷新应用才能看到变化

## 技术栈

- **后端**：Python 3 + http.server
- **前端**：原生 HTML + CSS + JavaScript
- **文件操作**：shutil (Python 标准库)

## 停止服务器

在终端按 `Ctrl+C` 即可停止服务器

## 故障排除

### 端口被占用
如果 8888 端口被占用，可以修改 `icon_manager.py` 中的 `PORT` 变量

### 图标不显示
检查 `public/uiicon/` 目录是否存在且包含图标文件

### 交换失败
确保有文件写入权限，且图标文件存在
