# 投喂功能 - 数据统计展示

## 功能概述

将投喂功能页面的专属徽章部分改造为数据统计展示，向用户展示他们在 lumostime 中的使用数据。

## 展示内容

在原有的问候语和支持者编号下方，新增 4 个数据统计卡片：

1. **记录时长** - 用户在 lumostime 中记录的总时长
2. **写的文字** - 用户在日志中写的总字数（包括标题和备注）
3. **记录瞬间** - 用户上传的图片总数
4. **一起走过** - 从第一次使用到现在的天数

## 技术实现

### 新增文件

- `src/services/userStatsService.ts` - 用户数据统计服务

### 修改文件

- `src/views/SponsorshipView.tsx` - 投喂功能页面
  - 导入 `userStatsService`
  - 添加 `userStats` 状态
  - 在验证成功后加载统计数据
  - 修改专属徽章部分的 UI，展示数据统计卡片

## 数据统计逻辑

### 记录时长
- 从 localStorage 读取所有日志数据
- 计算每条日志的 `endTime - startTime`
- 累加得到总时长（秒）
- 格式化为 "XX小时XX分钟" 或 "XX分钟"

### 写的文字
- 统计所有日志的 `title` 和 `note` 字段的字符数
- 累加得到总字数

### 记录瞬间
- 统计所有日志中 `images` 数组的长度
- 累加得到总图片数

### 一起走过
- 找到最早的日志记录日期
- 计算从该日期到今天的天数

## UI 设计

- 保持原有的问候语和支持者编号展示
- 在下方添加 2x2 网格布局的数据卡片
- 每个卡片包含：
  - 标题（小字，灰色）
  - 数值（大字，加粗）
- 卡片使用半透明白色背景，带有毛玻璃效果
- 整体保持与原有设计风格一致

## 使用场景

用户在验证投喂码后，可以看到自己在 lumostime 中的使用数据统计，增强用户的成就感和归属感。
