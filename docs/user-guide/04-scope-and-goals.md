# 领域与目标

## 什么是 Scope？
Scope 是跨分类的维度，用于从更高层面聚合时间：
- **能量流向**："写代码"、"写作" → **Output**；"看书"、"听播客" → **Input**
- **技能维度**："背单词"、"看美剧" → **English**

## 设定目标
在 **Scope 详情页** 设定周期性目标，支持 5 种指标：
- **时长目标**：累计投入多少小时
- **有效时长**：专注度加权的时长
- **任务数量**：完成多少个关联任务
- **活跃天数**：坚持打卡多少天
- **时长上限**：限制"坏习惯"的时间

系统自动统计关联记录更新进度。
