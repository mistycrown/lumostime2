# 标签管理

## 二级分类结构
- **Category（大分类）** → **Activity（具体活动）**
- 点击活动进入详情页，可编辑名称、颜色、图标

## 🔑 关键字功能
为活动添加**关键字**，用于分析和筛选，也可用于在补录时快速匹配选择标签：
1. 进入标签详情页 → **细节** 选项卡
2. 在 **Keywords** 区域添加关键字
3. 切换到 **Keywords** 选项卡查看：
   - **关键字热力图**：按关键字着色的日历视图
   - **关键字分组列表**：按关键字聚合的记录

## 🔗 自动关联规则
设置活动与领域的自动关联：
1. 进入 **设置** → **自动关联规则**
2. 添加规则：选择活动 → 选择要关联的领域
3. 之后补录该活动，会提醒是否关联到指定领域
