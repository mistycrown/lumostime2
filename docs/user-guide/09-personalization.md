# 个性化设置

## 关于投喂解锁

如果你喜欢这个应用，欢迎投喂本 mo 一个小鱼干 🐟

投喂后，你将收到来自本 mo 的小礼物 🎁

可以用它解锁**自定义主题设置**，包括：
- 更换背景图片
- 更换应用图标
- 更换导航栏装饰
- 使用主题方案
- 设置时间小友

## 如何解锁？

1. 进入 **设置** → **投喂功能**
2. 输入兑换码
3. 点击「解锁功能」按钮
4. 验证成功后即可使用个性化设置

**如何获取兑换码？**

详细说明请查看：[投喂功能说明文档](https://my.feishu.cn/wiki/QdlZw1vVai8DJakKvOzclKQ0nPk)

**想看看主题效果？**

查看各种主题的效果预览：[主题效果预览](https://my.feishu.cn/wiki/NlLSwoz7cidKm4kPqeqcjsQHnKe)

## 个性化功能

解锁后，可以使用以下功能：

### 1. 主题方案

一键应用预设的主题方案，包含：
- UI 主题
- 配色方案
- 背景图片
- 导航栏装饰
- 时间小友

你也可以创建自定义方案，保存自己喜欢的配置组合。

### 2. 时间小友

在界面中显示可爱的时间小友角色。时间小友会陪伴你记录时间，增添趣味。

### 3. 导航栏背景图片

更换导航栏的装饰图片。更换后，打开调试工具，根据设备屏幕调整装饰图片的位置。

### 4. 应用图标

更换应用启动器的图标样式（即手机桌面上显示的应用图标）。

注意事项：
- 更换后需要重启应用才能看到效果
- 如需更新应用，需先切换回默认图标，才能正常覆盖安装

### 5. UI 图标

更换应用内标签、分类、领域等的图标样式（即时间线、统计页面中显示的小图标）。目前提供 12 种主题风格。

注意事项：
- 首次从默认主题切换到自定义主题时，系统会自动为所有标签生成对应的 UI 图标
- 切换主题后，所有标签的图标样式会统一更换为新主题风格
- 可以在标签详情页单独调整某个标签的 UI 图标
