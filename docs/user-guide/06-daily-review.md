# 回顾与复盘

## 什么是回顾？

回顾帮助我们从数据中提取洞察。

通过回顾，你可以：
- 看见时间的真实流向
- 发现问题和改进空间
- 记录成长和变化
- 与过去的自己对话

## 在哪里回顾？

### 档案页面
**档案页面**汇总了所有回顾：
- **每日回顾**（Daily Review）
- **每周回顾**（Weekly Review）
- **每月回顾**（Monthly Review）

### 脉络页面
在 **脉络** 页面底部，会自动出现回顾入口：

**每日回顾：**
- 每天都会出现「准备好开始回顾了吗」

**每周回顾：**
- 每周日会出现周回顾入口

**每月回顾：**
- 每月最后一天会出现月回顾入口

**出现时间：**
- 在 **设置** → **偏好设置** 中设定回顾时间点
- 到达设定时间后，脉络时间线上会出现回顾节点
- 点击即可创建回顾

## 回顾的四个维度

每个回顾（日/周/月）都包含四个 Tab：

### 1. 日课（Check）

记录每日习惯打卡项目。

**功能：**
- 从模板导入打卡项（如运动、阅读、冥想）
- 每天勾选完成情况
- 在数据统计视图中查看打卡天数

**模板设置：**
- 在 **设置** 页面添加自定义日课模板

**重要规则：**

1. **数据固定机制**
   - 生成回顾后，日课数据即被固定
   - 修改模板内容无法同步更新已生成的日课数据
   - 如需应用最新模板，在日课页面底部点击「重新从模板导入」

2. **单个日课项**
   - 每天可以手动添加单个日课项，用于临时提醒或记录
   - 注意：单个日课项不会纳入统计，只有从模板导入的项才会参与统计汇总

3. **统计规则**
   - 统计是自下而上汇总的，请尽量不要修改日课名称
   - 如果修改了名称（例如从「喝水」改为「喝水八杯」），系统会将其识别为两个不同的项目，导致统计数据无法连续汇总

### 2. 数据（Stats）

查看时间统计数据。

**功能：**
- 标签分布：时间都花在哪些活动上了？
- 待办统计：推进了哪些任务？
- 领域分布：在不同维度上的投入情况如何？
- 支持多种视图：饼图、折线图、日程图

### 3. 引导（Guide）

结构化复盘，通过问题引导思考。

**功能：**
- 填写预设模板（如 Keep/Problem/Try）
- 在设置中自定义模板和问题
- 切换阅读模式，排版更清晰
- 支持问答题、选择题、评分题三种类型

### 4. 叙事（Narrative）

深度日记，记录感受和思考。

**功能：**
- **AI 共创**：基于时间记录和统计数据，AI 自动生成连贯的叙事日志
- **手动撰写**：新建空白叙事，自由书写
- **个性化设置**：在 **设置** → **AI 叙事设置** 中配置画像和提示词，让 AI 更懂你

**9 种预设模板：**

1. **赛博黄历** 📜
   - 基于天干地支与五行生克的赛博运势推演
   - 适用：每日回顾

2. **哲学命题** 🌟
   - 现代哲学家视角，对经历进行概念升维
   - 适用：每日/每周/每月回顾

3. **每日简报** 📅
   - 客观务实，数据导向的每日总结
   - 适用：每日回顾

4. **极简复盘** 📊
   - 剥离细节，关注总量与趋势的阶段性报告
   - 适用：每周/每月回顾

5. **严格导师** ⚔️
   - 犀利、不留情面的漏洞审视与鞭策
   - 适用：每日/每周/每月回顾

6. **正念叙事** 🍃
   - 温柔的心理疗愈，引导觉察与接纳
   - 适用：每日/每周/每月回顾

7. **成长教练** 🚀
   - 行为设计与长期主义的深度复盘
   - 适用：每日/每周/每月回顾

8. **苏维埃** 🚩
   - 以革命导师的钢铁意志审视生产战线
   - 适用：每日/每周/每月回顾

9. **树洞伴侣** 🧞‍♂️
   - 整理碎片想法，提供温暖洞察的树洞
   - 适用：每日回顾
