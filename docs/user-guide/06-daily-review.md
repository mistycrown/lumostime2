# 每日回顾与 AI 叙事

每天结束时，点击 Timeline 的底部"准备好开始回顾了吗"，进入每日回顾。

## 📊 数据概览 
首先查看今日的数据统计：
- 标签分布：时间都花在哪些分类上了？
- 待办统计：今天推进了哪些任务？
- 领域分布：在不同维度上的投入情况如何？

## 📅 结构化复盘 
切换到 **引导 (Guide)** 标签页进行结构化复盘：
- **填写模板**：回答预设问题（如 Keep/Problem/Try），支持在设置中自定义模板
- **阅读模式**：填写后可切换为阅读模式，排版更为清晰

## ✨ AI 叙事日志 
切换到 **叙事 (Narrative)** 标签页，体验深度日记：
- **与 AI 共创**：点击按钮，AI 将基于你今天的**时间记录**和**统计数据**，自动生成一段连贯、深刻的叙事日志
- **手动撰写**：也可以选择新建空白叙事，自由书写
- **个性化设置**：在 **Settings** → **AI Narrative Settings** 中配置你的画像和提示词，让 AI 更懂你的风格

## 🕰️ 往年今日 
如果系统检测到你在往年的今天有记录，Timeline 底部会出现提示：
- 点击 **时空隧道** 按钮
- 穿越回过去，回顾去年的自己在做些什么
