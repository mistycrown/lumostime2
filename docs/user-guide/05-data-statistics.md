# 数据统计

## 全局统计 
- **饼图视图**：日/周/月/年 四个维度的时间分配
- **矩阵视图**：周度活动打卡概览
- **日程视图**：一日/一周的可视化时间轴
- **全屏模式**：沉浸式数据分析

## 专项分析
在标签/领域/待办详情页查看：
- **投入热力图**：类似 GitHub Contribution 的日历热力图
- **专注热力图**：按专注分着色的热力图
- **专注堆积图**：不同专注等级的时长趋势
- **领域分布**：时间在各领域的分配比例

## 📤 导出报告
- **Timeline 页面**：导出日报（含时间段、分类、备注、专注分）
- **Stats 页面**：导出统计摘要
