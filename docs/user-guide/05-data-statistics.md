# 数据统计

## 为什么需要数据统计？

记录时间只是第一步，看见数据才能产生洞察。

数据统计帮你回答：
- 我的时间都去哪儿了？（分布）
- 我坚持得怎么样？（连续性）
- 我在持续进步吗？（趋势）
- 我的专注质量如何？（深度）

## 全局统计

在 **统计** 页面查看整体时间分布，支持 5 种视图：

### 1. 饼图视图（Pie Chart）
查看时间分配比例，支持四个维度：
- **日**：今天的时间分布
- **周**：本周的时间分布
- **月**：本月的时间分布
- **年**：今年的时间分布

### 2. 矩形矩阵视图（Matrix）
展示一周内各个活动的发生情况。

一眼看出本周哪些活动做了，哪些活动被忽略了。

### 3. 折线视图（Line Chart）
展示时间投入的趋势变化，支持：
- **周视图**：过去 7 天的趋势
- **月视图**：过去 30 天的趋势

### 4. 日程视图（Schedule）
可视化时间轴，展示一天/一周的时间安排：
- **日视图**：今天的时间块分布
- **周视图**：本周每天的时间块
- **月视图**：本月每天的时间块

### 5. 日课打卡视图（Check）
追踪每日习惯打卡情况，基于 Daily Review 的 Check 项。

统计周、月、年的习惯打卡情况。

## 专项分析

在 **标签/领域/待办详情页** 查看针对性分析：

### 1. 热力图（Heatmap）
展示该标签/领域/待办的投入时间。

### 2. 画廊视图（Gallery）
展示时间记录中关联的图片，形成视觉日志。

### 3. 关键字分析
在标签详情页中切换到关键字视图，查看关键字分布。

### 4. 专注统计条形图
展示不同专注等级的时长分布。

### 5. 交叉分析
在 **关联** 选项卡中查看：
- **标签详情页**：该标签关联了哪些待办和领域
- **领域详情页**：该领域关联了哪些待办和标签

## 报告功能

### 1. 统计导出
在 **统计** 页面点击导出按钮：
- 导出饼状图的统计摘要
- 包含时间分布、占比等数据

### 2. 日报导出
在 **脉络** 页面点击导出按钮：
- 导出当天的时间记录
- 包含时间段、活动、备注、专注分
