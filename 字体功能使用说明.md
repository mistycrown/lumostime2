# 字体功能使用说明

## ✅ 已完成

我已经为你的应用添加了完整的字体切换功能！所有代码已经编写完成并通过编译测试。

## 📍 功能位置

进入应用后：
1. 点击底部导航栏的"投喂功能"

## 🎨 可用字体

目前预设了 4 款字体：

1. **默认** - 系统默认字体（Noto Serif SC）
2. **霞鹜文楷** - 温暖的手写风格 ⭐ 推荐
3. **思源宋体** - 优雅的传统宋体
4. **阿里巴巴普惠体** - 现代简洁风格

## 📥 下载字体文件

### 方法一：直接下载（推荐）

我已经为你准备好了下载链接，请按照以下步骤操作：

#### 1. 霞鹜文楷（推荐优先下载）
```
下载地址：https://github.com/lxgw/LxgwWenKai/releases
1. 打开链接，找到最新版本
2. 下载 Assets 中的字体文件包
3. 解压后找到 woff2 格式的文件：
   - LXGWWenKai-Regular.woff2
   - LXGWWenKai-Bold.woff2
4. 放到：public/fonts/lxgw-wenkai/ 目录
```

#### 2. 思源宋体
```
下载地址：https://github.com/adobe-fonts/source-han-serif/releases
1. 找到 "SourceHanSerifSC" (简体中文版本)
2. 下载 woff2 格式：
   - SourceHanSerifSC-Regular.woff2
   - SourceHanSerifSC-Bold.woff2
3. 放到：public/fonts/source-han-serif/ 目录
```

#### 3. 阿里巴巴普惠体
```
下载地址：https://www.alibabafonts.com/
1. 在官网下载普惠体
2. 转换为 woff2 格式（或直接下载 woff2）：
   - AlibabaPuHuiTi-Regular.woff2
   - AlibabaPuHuiTi-Bold.woff2
3. 放到：public/fonts/alibaba-puhuiti/ 目录
```

### 方法二：使用字体转换工具

如果你下载的是 ttf 或 otf 格式，可以使用在线工具转换：
- https://cloudconvert.com/ttf-to-woff2
- https://www.fontsquirrel.com/tools/webfont-generator

## 📂 文件放置位置

确保文件名和路径完全一致：

```
public/fonts/
├── lxgw-wenkai/
│   ├── LXGWWenKai-Regular.woff2    ← 放这里
│   └── LXGWWenKai-Bold.woff2       ← 放这里
├── source-han-serif/
│   ├── SourceHanSerifSC-Regular.woff2  ← 放这里

└── alibaba-puhuiti/
    ├── AlibabaPuHuiTi-Regular.woff2    ← 放这里
    └── AlibabaPuHuiTi-Bold.woff2       ← 放这里
```

## 🚀 测试步骤

1. **放置字体文件**（按照上面的说明）

2. **启动应用**
   ```bash
   npm run dev
   ```

3. **打开浏览器开发者工具**（F12）
   - 切换到 Network 标签
   - 刷新页面
   - 搜索 `.woff2`
   - 确认字体文件加载成功（状态码 200）

4. **测试字体切换**
   - 进入"投喂功能" → "字体" Tab
   - 点击不同的字体选项
   - 观察页面文字是否立即改变

## ⚠️ 注意事项

### 文件命名
- 文件名必须完全一致（区分大小写）
- 必须是 `.woff2` 格式
- 不要修改文件名

### 常见问题

**Q: 字体没有生效？**
- 检查文件名是否正确
- 检查文件是否放在正确的目录
- 打开浏览器控制台查看是否有 404 错误
- 清除浏览器缓存后重试

**Q: 只想用一款字体可以吗？**
- 可以！只下载你想要的字体即可
- 其他字体会自动回退到默认字体

**Q: 字体文件太大？**
- woff2 已经是最优压缩格式
- 每个字体约 2-5MB，总共 8-10MB
- 这是一次性加载，后续会缓存
