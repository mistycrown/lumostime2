# ShareCard 组件

## 功能概述
ShareCard 组件用于将时间记录转换为精美的排版图片卡片，方便用户在社交媒体上分享。

## 设计理念
- **极简主义**：减少卡片内部要素，保持视觉空旷
- **纯粹文字**：仅保留活动标签，移除领域标签
- **优雅排版**：5种主题色彩 + 5种布局模板

## 组件结构

```
src/components/ShareCard/
├── constants.ts          # 主题和模板配置
├── types.ts             # TypeScript 类型定义

└── README.md           # 本文档
```

## 使用方式

### 1. 入口
在记录详情页面（AddLogModal）中，点击 "Share" 按钮即可打开分享页面。

### 2. 功能特性
- **5种主题色彩**：水墨黑、朱砂红、竹青、靛蓝、极简白
- **5种布局模板**：经典杂志、竖排古韵、现代分割、胶片时刻、留白便签
- **固定画幅比例**：3:4（适合社交媒体分享）
- **高清导出**：3x 像素密度的 PNG 格式
- **极简设计**：仅显示必要信息，保持视觉空旷

### 3. 数据映射
Log 数据自动映射到卡片内容：
- `note` → 正文内容
- `images` → 配图
- `startTime` → 日期
- `Activity` → 活动标签（纯文字，无图标）

## 技术实现

### 依赖库
- `html-to-image`: 将 DOM 转换为图片
- `lucide-react`: 图标库

### 核心流程
1. 用户在记录详情页点击 Share 按钮
2. 打开 ShareView 页面，显示预览和编辑区域
3. 用户选择主题和模板
4. 点击保存按钮，将 DOM 转换为高清 PNG 图片
5. 自动下载到本地

## 模板说明

### 1. 经典杂志 (magazine-classic)
- 居中排版，优雅大方
- 适合大多数场景
- 首字母放大效果
- 顶部显示活动标签和日期

### 2. 竖排古韵 (vertical-poetry)
- 传统竖排文字
- 适合诗歌和短文
- 中国风设计
- 极简元素，突出文字

### 3. 现代分割 (modern-split)
- 左文右图布局
- 理性与感性的平衡
- 适合多图展示
- 底部显示活动标签

### 4. 胶片时刻 (film-story)
- 拍立得风格
- 大留白设计
- 怀旧质感
- 极简信息展示

### 5. 留白便签 (minimal-note)
- 极简风格
- 专注于文字本身
- 适合短文和引言
- 最大化留白空间

## UI 优化

### 主题选择器
- 5个圆形色块均匀分布
- 圆圈尺寸：40px × 40px
- 选中状态：ring 效果 + 放大
- 响应式布局，确保所有圆圈完整显示

### 视觉简化
- 移除所有 UI 图标（UI Code、UI Sleep 等）
- 仅保留活动标签的纯文字
- 移除领域标签
- 减少装饰元素，增加留白
